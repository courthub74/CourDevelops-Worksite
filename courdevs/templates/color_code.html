Disable Fields By CheckBox


  1. Color Code selection output by Radiodial
  2. 'Onsite' value is color coded yellow upon selection
    'Instudio' is color coded purple upon selection
    You can plug in ANY colors or other CSS settings for this sequence
    Le CSS

    <style type="text/css">
    .goldy {
    background-color: #FFD700;
    }
    .prince {
    background-color: #DDA0DD;
    }
    </style>
    Le JavaScript

    <script type="text/javascript">
    var colorElements = document.querySelectorAll('.color');
    colorElements.forEach(element => {
    if (element.innerHTML == "OnSite") {
    element.classList.add('goldy');
    } else if (element.innerHTML ==
    "InStudio") {
    element.classList.add('prince');
    }
    });
    </script>
