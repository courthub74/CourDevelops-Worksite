Database (Models)


  1. Create it in models.py
  2. you create a CLASS like below:
    you can call it whatever you need. In this case, it's the 'Items' class

    class Items(models.Model):
    item1 = models.CharField(max_length=#)
    item2 = models.BooleanField(default=False)
    item3 = models.EmailField()
    item4 = models.IntegerField()

    def __str__(self):
    return self.item1     <-- always do the first item on the class list

  3. NEXT go to admin.py
  4. from .models import Items <-- or from ALL models import Items class
    now register your model
    admin.site.register(Items)

  5. THEN go to cmdline and type:

    • python manage.py makemigrations

      • then....

    • python manage.py migrate

    NOW you can go into admin and add data to newly created table
