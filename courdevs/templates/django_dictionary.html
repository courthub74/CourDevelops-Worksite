Context Dictionary


Transport Info from models.py


  1. in views.py
  2. Set up the 'Home' page

    def home(request):
    my_name = "Court"
    return render(request, 'home.html', {'name': my_name})

  3. in home.html
  4. Transport the variable 'my_name' by calling the 'name' dictionary from above

    (% extends 'home.html' %)
    (% block content %)
    My name is: (( name ))
    (% endblock %)
    The output on home.html will read:
    My name is: Court

  5. Change the output by just changing the value
  6. in the dictionary you can change anything by just changing the value
    def home(request):
    my_name = "Court"
    number = 4 + 9
    return render(request, 'home.html', {'name': number})
    We've changed the value from my_name to number
    The output from:

    (% extends 'home.html' %)
    (% block content %)
    My name is: (( name ))
    (% endblock %)
    will now read what's in the number variable:
    4 + 9 which equals 13

    So the home.html screen will say:
    My name is: 13

  7. You can also have more than one dictionary
  8. BELOW we have firstname and lastname
    def home(request):
    firstname = "Court"
    lastname = "Jacobs"
    return render(request, 'home.html', {'firstname': firstname, 'lastname': lastname})

  9. in home.html (more than one dictionary)
  10. You can have:

    (% extends 'home.html' %)
    (% block content %)
    My first name is: (( firstname ))
    My last name is: (( lastname ))
    (% endblock %)
    The output on home.html will read:
    My first name is: Court
    My last name is: Jacobs
