Cloning aRepo


  1. Cloning GitHub Repo
  2. mkdir (project folder)
    cd into folder of clone location
    install virtual environment (if needed)(pip install virtualenv)
    Create virtual environment (virtualenv venv)
    turn on virtual environment (. venv/Scripts/activate)
    install Django (pip install django)
    Clone the project into the location (project folder)
    install dependencies:

    • gunicorn (pip install gunicorn)
    • django-heroku (pip install django-heroku) (if pushing to heroku)
    • python-decouple (pip install python-decouple)

    copy the .env file

    from the local one on your computer
    or you will have to get the secret key from whoever owns the repository
