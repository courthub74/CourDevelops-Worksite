Basic Structure


  1. Initial Setup and Installation
  2. mkdir (project)
    in (project) install flask (pip install flask)
    open text editor and attach the folder (project)
    create a new file within (project) called (new file) --[save as]
    then enter below in (new file):

    from flask import Flask


    app = Flask(__main__)



    if __name__ == "__main__":
    app.run(debug=True)
    That sets up the Basic Structure
    NEXT you add pages like so:

    from flask import Flask

    app = Flask(__main__)

    @app.route('/')
    def home():
    return render_template("home.html")
    THIS sets up the 'Home' page

    @app.route('/about')
    def about():
    return render_template("about.html")
    THIS sets up the 'About' page

    @app.route('/pricing')
    def pricing():
    return render_template("pricing.html")
    THIS sets up the 'Pricing' page


    if __name__ == "__main__":
    app.run(debug=True)

    And so forth ....
    To Run the app:
    • from (cmdline) cd into (project)
    • type: python (new file)

    You can then create links to the different pages inside the HTML
