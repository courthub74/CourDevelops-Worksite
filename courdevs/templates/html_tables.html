  2. BELOW makes an HTML table:
    <table>
    <tr>
    <th scope="col">Saturday</th>
    <th scope="col">Sunday</th>
    </tr>
    <tr>
    <th scope="row">Temperature</th>
    <td>73</td>
    <td>81</td>
    </tr>
    <table>

    <tr> - Table Row

    <th> - Table Heading

    <td> - Table Data
    NOTE: the SCOPE attribute, which can take one of two values here:

    row - this value makes it clear that the heading is for a ROW.
    col - this value makes it clear that the heading is for a COLUMN
    Bear in mind we haven't made the boarder yet
    BELOW makes a table boarder
    <table border="1"> <--the integer "1" represents the thickness of the border
    <tr>
    <td>73</td>
    <td>81</td>
    <td>61</td>
    <td>31</td>
    <td>21</td>
    <td>91</td>
    <td>81</td>
    </tr>
    </table>

    Outcome BELOW the numbers are blocked with boarders
    73 81 61 31 21 91 81
    CSS version reads like this:

    table, td {
    border: 1px solid black;,
    }
    SO the CSS would work on the HTML ABOVE
    BELOW is an example of columnspan:
    <table border="1">
    <tr>
    <th>Monday</th>
    <th>Tuesday
    <th>Wednesday
    </tr>
    <tr>
    <td colspan="2">Out of Town
    <td>Back in Town
    </tr>
    </table>

    Monday Tuesday Wednesday
    Out of Town
    		 Back in Town
    * HERE 'Out of Town' covers both Mon and Tue
    BELOW is an example of rowspan:
    <table border="2">
    <tr> <!-- TableRow 1 -->
    <th>
    <th>Saturday
    <th>Sunday
    </tr>
    <tr> <!-- TableRow 2 -->
    <th>Morning
    <td rowspan="2">Work
    <td rowspan="3">Relax
    </tr>
    <tr> <!-- TableRow 3 -->
    <th>Afternoon
    </tr>
    <tr> <!-- TableRow 4 -->
    <th>Evening
    <td>Dinner
    </tr>
    </table>

    Saturday Sunday
    Morning Work Relax
    Afternoon
    Evening Dinner
    * SO its scheduled to WORK Saturday Morning and Afternoon AND relax all of Sunday
