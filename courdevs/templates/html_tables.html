  2. BELOW makes an HTML table:

    <table>
    <tr>
    <th scope="col">Saturday</th>
    <th scope="col">Sunday</th>
    </tr>
    <tr>
    <th scope="row">Temperature</th>
    <td>73</td>
    <td>81</td>
    </tr>
    <table>

    <tr> - Table Row

    <th> - Table Heading

    <td> - Table Data
    NOTE: the SCOPE attribute, which can take one of two values here:

    row - this value makes it clear that the heading is for a ROW.
    col - this value makes it clear that the heading is for a COLUMN
    Bear in mind we haven't made the boarder yet
